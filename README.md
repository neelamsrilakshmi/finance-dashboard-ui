
# Finance Dashboard UI

A simple and interactive finance dashboard built with React for the Frontend Developer Intern assignment.

## Features
- Dashboard summary cards
- Balance trend chart
- Spending breakdown chart
- Transactions table
- Search and filters
- Role-based UI (Viewer / Admin)
- Admin add and delete transaction
- Insights section
- Responsive design

## Tech Stack
- React
- Vite
- Recharts
- CSS

## Run Locally

```bash
npm install
npm run dev
